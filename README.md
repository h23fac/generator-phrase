# 🧠 Mini Almacén de Frases

Este proyecto tiene como objetivo desarrollar una aplicación web básica que permita gestionar frases (crear, visualizar, eliminar y editar), mientras se aprenden y refuerzan los fundamentos del desarrollo frontend.

---

## 🎯 Objetivo del Proyecto

Construir una aplicación funcional desde cero utilizando HTML, CSS y JavaScript puro, enfocándose en:

- Manipulación del DOM
- Manejo de eventos
- Gestión de estado (datos en memoria)
- Renderizado dinámico
- Persistencia de datos (localStorage)

Este proyecto evolucionará progresivamente hasta convertirse en una aplicación más completa, incluso integrando backend en etapas posteriores.

---

## 🚀 Funcionalidades Iniciales

- Agregar nuevas frases
- Mostrar lista de frases
- Eliminar frases
- Validar que no se ingresen frases vacías

---

## 🔧 Funcionalidades a Desarrollar (Evolución)

### 🟡 Nivel 1 (Base)
- Input para agregar frases
- Botón de agregar
- Lista dinámica de frases
- Eliminación de frases

---

### 🟠 Nivel 2 (Intermedio)
- Editar frases
- Marcar frases como favoritas ⭐
- Buscador de frases
- Mejor estructura del código (funciones separadas)

---

### 🔵 Nivel 3 (Persistencia)
- Guardar frases en `localStorage`
- Cargar frases automáticamente al iniciar la app

---

### 🟣 Nivel 4 (Preparación para Backend)
- Simulación de consumo de API (fetch)
- Manejo de errores básicos
- Separación de lógica y presentación

---

### 🔴 Nivel 5 (Futuro - Backend)
- Conexión a una API real
- CRUD completo con backend
- Base de datos
- Sistema de usuarios (login básico)
